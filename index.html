Quick Start

# Convert a system-level Markdown to HTML
node system-md-to-html.js doc/System_Arch.md

# Convert all system docs
node system-md-to-html.js --all

# Generate index page
node system-md-to-html.js --index "Project" "Description"

# Validate
node validate-html.js --all
